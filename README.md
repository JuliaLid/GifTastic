# Magical Gifs

## Overview: 
Use the GIPHY API to make a dynamic web page that populates with gifs based on user's choice. Call the GIPHY API and use JavaScript and jQuery to change the HTML.

## Technology 
* HTML
* CSS Grid layout
* JavaScript 
* jQuery (AJAX)
